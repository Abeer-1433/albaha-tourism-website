# موقع منطقة الباحة السياحي

موقع ويب حديث لعرض المعالم السياحية في منطقة الباحة، مبني باستخدام Next.js و Tailwind CSS.

## المميزات

- ✅ تصميم عربي بالكامل مع دعم RTL
- ✅ واجهة مستخدم حديثة وجذابة
- ✅ متجاوب مع جميع أحجام الشاشات
- ✅ أربع صفحات رئيسية:
  - الصفحة الرئيسية مع قسم Hero يدعم الفيديو
  - صفحة ذي عين التراثية
  - صفحة غابة رغدان
  - صفحة منتزه جدر

## بدء التشغيل

1. تأكد من تثبيت Node.js (الإصدار 16 أو أحدث)

2. تشغيل خادم التطوير:
```bash
npm run dev
```

3. افتح المتصفح على: [http://localhost:3000](http://localhost:3000)

## البناء للإنتاج

```bash
npm run build
npm start
```

## الهيكل

```
/
├── pages/              # صفحات الموقع
│   ├── index.tsx      # الصفحة الرئيسية
│   ├── thee-ain.tsx   # صفحة ذي عين
│   ├── raghdan-park.tsx # صفحة غابة رغدان
│   └── jodur-park.tsx # صفحة منتزه جدر
├── components/        # مكونات React
│   ├── Layout.tsx     # القالب الرئيسي
│   ├── Header.tsx     # رأس الصفحة والقائمة
│   ├── Footer.tsx     # ذيل الصفحة
│   └── Hero.tsx       # قسم البطل
├── public/           # الملفات الثابتة
│   └── images/       # الصور والشعار
└── styles/           # ملفات التنسيق
    └── globals.css   # CSS العام
```

## إضافة فيديو للصفحة الرئيسية

لإضافة فيديو خلفية للصفحة الرئيسية:
1. ضع ملف الفيديو في مجلد `public/videos/`
2. قم بتعديل اسم الملف في `pages/index.tsx` في خاصية `videoSrc` للمكون `Hero`

## التخصيص

- الألوان: يمكن تعديلها في `tailwind.config.js`
- الخطوط: تم استخدام خطي Cairo و Tajawal من Google Fonts
- المحتوى: يمكن تعديل النصوص مباشرة في ملفات الصفحات

## التقنيات المستخدمة

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS 3
- Google Fonts (Cairo, Tajawal)
